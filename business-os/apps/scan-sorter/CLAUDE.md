# スキャン自動仕分け — アプリ設計書

## 概要
ScanSnapでスキャンした書類を自動でOCR→分類→リネーム→フォルダ振り分け→税理士共有する。

## 全体フロー

```
ScanSnap
  ↓ ScanSnapの設定で自動保存
Googleドライブ「スキャン受信BOX」フォルダ
  ↓ GAS トリガー（5分おき or ファイル追加検知）
OCR処理（Google Vision API）
  ↓ テキスト抽出
AI分類（Claude API or ルールベース）
  ↓ 書類種別判定 + 科目推定
リネーム
  ↓ 日付_取引先_金額_科目.pdf
フォルダ振り分け
  ↓ 科目別フォルダに移動
  ├→ 税理士共有対象 → 税理士共有フォルダにコピー
  └→ マネーフォワード連携対象 → MFマッチングキューに追加
```

## 書類分類ルール

### 分類カテゴリ
| カテゴリ | 例 | 税理士共有 |
|----------|-----|-----------|
| 領収書 | レシート、領収書 | ○ |
| 請求書 | 各種請求書 | ○ |
| 契約書 | 賃貸契約、業務委託契約 | ○ |
| 通帳 | 銀行通帳コピー | ○ |
| 納税通知 | 固定資産税、所得税 | ○ |
| 保険 | 火災保険、地震保険 | ○ |
| 物件資料 | 物件概要書、重要事項説明書 | × |
| その他 | 分類不能 | △（要確認） |

### リネーム規則
```
{YYYYMMDD}_{取引先}_{金額}円_{科目}.pdf
例: 20260315_ヤマダ電機_12800円_消耗品費.pdf
例: 20260301_みずほ銀行_通帳_202603.pdf
```

### フォルダ構成（Googleドライブ）
```
経理/
├── スキャン受信BOX/          ← ScanSnapの保存先
├── 処理済み/
│   ├── 2026/
│   │   ├── 03/
│   │   │   ├── 領収書/
│   │   │   ├── 請求書/
│   │   │   ├── 契約書/
│   │   │   ├── 通帳/
│   │   │   └── その他/
│   │   └── ...
│   └── ...
├── 税理士共有/               ← 税理士のGoogleアカウントと共有
│   ├── 2026/
│   │   ├── 03/
│   │   └── ...
│   └── ...
└── 分類エラー/               ← 分類できなかったもの（手動確認用）
```

## スプレッドシート設計

### スキャン仕分けログ
| カラム | 型 | 説明 |
|--------|-----|------|
| ファイルID | 文字列 | Google DriveのファイルID |
| 元ファイル名 | 文字列 | |
| 新ファイル名 | 文字列 | リネーム後 |
| スキャン日時 | 日時 | |
| 処理日時 | 日時 | |
| 書類種別 | 文字列 | 領収書/請求書/... |
| 取引先 | 文字列 | OCRから抽出 |
| 金額 | 数値 | OCRから抽出 |
| 日付 | 日付 | 書類上の日付 |
| 科目 | 文字列 | 推定された勘定科目 |
| 移動先フォルダ | 文字列 | |
| 税理士共有 | boolean | |
| MF連携 | boolean | マネーフォワードに送信済みか |
| 信頼度 | 数値 | OCR/分類の信頼度(0-100) |
| 手動確認要 | boolean | 信頼度が低い場合TRUE |
| ステータス | 文字列 | 処理中/完了/エラー/要確認 |

### 科目マスタ
| カラム | 型 | 説明 |
|--------|-----|------|
| 科目コード | 文字列 | |
| 科目名 | 文字列 | 消耗品費、交通費、etc |
| キーワード | 文字列 | 分類用キーワード（カンマ区切り） |
| デフォルト税率 | 数値 | 10%, 8%, 0% |
| 税理士共有対象 | boolean | |

## 技術的な注意点
- ScanSnap → Googleドライブ連携は ScanSnap Home の「クラウド保存」機能を使用
- GAS の時間制限（6分）内で処理できるファイル数に制限あり → バッチ処理
- Google Vision API は無料枠あり（月1,000リクエスト）→ 超過分は課金
- OCR精度が低い場合は手動確認フラグを立てる
