## データフロー

### 民泊の1日の流れ
```
[前日夜]
  黒子が翌日のスケジュール確認
  → スタッフ未確定なら🔴アラート

[当日朝 6:00]
  黒子が朝ブリーフィング配信

[チェックアウト時刻]
  BEDS24から予約ステータス更新（将来）
  → 清掃開始通知をスタッフに送信

[清掃中]
  スタッフがチェックリスト実施（写真付き）
  → コインランドリー使用記録入力

[清掃完了]
  チェックリスト送信
  → オーナーに完了報告自動送信
  → シフト実績に「完了」記録

[チェックイン前]
  ゲストにチェックイン案内自動送信（将来、BEDS24連携後）
```

### 経理の月次フロー
```
[日々]
  ScanSnap → Googleドライブ → 自動仕分け → 税理士共有

[月末]
  請求書自動生成 → スタッフ確認 → PDF生成 → オーナーに送付

[月初]
  前月の仕訳データ整理
  マネーフォワードとの照合
  月次レポート生成
```

## 認証・アクセス制御

| ユーザー | アクセス範囲 | 認証方法 |
|----------|-------------|----------|
| オーナー | 全機能 | Googleアカウント（GAS Web App） |
| スタッフ | シフト確認、チェックリスト、請求書確認 | URLパラメータ or メール認証 |
| 税理士 | 共有フォルダのみ | Googleドライブ共有 |
